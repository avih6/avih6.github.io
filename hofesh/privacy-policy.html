2. Information Collection and Use

Personal Data

The App does not collect any personally identifiable information (PII) such as names, email addresses, or phone numbers.

Third-Party Analytics and Performance

The app uses third-party services that may collect information used to identify you anonymously for analytics and stability purposes:

  • Google Analytics for Firebase: To understand how the app is used.
  • Google Analytics: Used on this website to measure anonymous traffic data (such as page views) to help improve the service.
  • Firebase Crashlytics: To receive technical reports in case of an app crash.
  • Firebase Performance Monitoring: To collect performance metrics and identify technical issues to improve app responsiveness.

External API (Hebcal)

The application communicates with Hebcal.com to retrieve holiday data. This communication is processed ephemerally to provide real-time updates and core functionality. This process does not involve the storage of personal data or the tracking of user identity.

3. Log Data

In the event of an error, the application collects technical information through third-party products called Log Data. This data is used solely for debugging purposes and includes the device hardware model, operating system version, and the time of the event.
